# AllJobs - Job Portal React App

A modern job portal web application built with **React**, **React Router**, and **Tailwind CSS**. Users can explore jobs, update profiles, and apply for positions.  

## Features

- Fully responsive UI with **Tailwind CSS**  
- **AllJobs Banner** with highlights and search bar  
- View, add, and update job listings  
- Integration with **REST API** (CRUD operations)  
- SweetAlert2 toast notifications for actions  
- Dynamic routing with **React Router v6**
